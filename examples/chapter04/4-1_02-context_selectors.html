Context Selectors

In this example we will look at several more advanced CSS selectors which allow us to specify the context in which an HTML entity must appear in order to have the style rule apply.

  • Menus from local restaurants
    1. La Casita
    2. Phoumy's Thai Cuisine
    3. The Brik
    4. Hepler's Big Cheese Pizza
  • University Bulletins/Catalogs
    1. Walla Walla University
    2. University of Louisville
    3. Iowa State University
    4. City University of Seattle

There are three types of selectors that we will look at. They are: selectors based on nesting, selectors based on attribute values, and additional pseudo-class selectors allowing us to insert content.
